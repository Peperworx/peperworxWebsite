The Peperworx game launcher is a platform that is currently being built to distribute games and other software, both from Peperworx, and other third party developers. On release, this game launcher will be published to the OpenSource community. This launcher will enable us and third party developers to publish applications to a cross-platform environment. The only foreseeable requirement for developers is to host the files and some of the data for their game, as well as multiplayer, as we do not currently have the resources to host this data for them.
